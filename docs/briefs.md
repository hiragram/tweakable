# RevenueCat Shipyard: Creator Contest - クリエイターブリーフ詳細

## 2. Eitan Bernath

**テーマ**: From saved recipe to dinner made（保存したレシピから実際の夕食へ）

### Notes
保存したレシピや料理動画を、アイデアの整理とスマートな買い物リストの生成によって、実際の食事に変えるアプリを作ってほしい。

### Transcript（文字起こし）
こんにちは。僕はEitan Bernath、23歳のシェフ、コンテンツクリエイター、料理本の著者で、Drew Barrymore Showの料理担当でもある。要するに、インターネットやテレビで料理して、みんなにキッチンに立ってもらうことを目指してる。

僕のアプリのアイデアは、最近みんな、特に料理本やソーシャルメディアでレシピを大量に保存してるけど、「見たもの」から「実際に作る」までのハードルが高いという問題に基づいてる。

2つのアイデアがある：

1つ目は料理本側で、僕のコレクションにある全ての料理本をアプリに登録できて、冷蔵庫に鶏肉、ブロッコリー、いくつかの材料があるとアプリに伝えると、その材料を使ったレシピが料理本の中にあるか教えてくれる。著作権の問題があるかもしれないけど、1つのアイデアだ。

もう1つは、作りたい動画やネット上のレシピのリンクをアプリに渡すと、それを実際に作るための買い物リストを自動生成してくれるツール。正直、「見た」から「テーブルに並んだ」までの過程で、人々が実際に料理するようになる手助けをするものなら何でもいい。買い物を可能にし、アイデアからキッチンでの実現までのプロセスをスムーズにするもの。何を作ってくれるか楽しみにしてる。

### キャリア・実績

#### 料理を始めた経緯

8歳の時に料理を始めた。食に夢中だった彼が母親に伝統的な料理をもっと試してほしいと頼んだところ、母親は「自分でやってみなさい」と言い、二人で一緒に料理を探求し始めた。初めて一人で作った料理は、缶詰の豆とトマトソースを温めてトルティーヤとチーズを添えたもの。

両親は教師の給料で育ったため海外旅行にはあまり行けなかったが、キッチンで中華料理、インド料理、メキシカン料理、イタリア料理などを作ることで「食を通じた旅」を経験させた。

#### まとめ

日本の食材（醤油、味噌、みりん）を使用したレシピや焼肉ソースのような日本風レシピは紹介しているが、日本への訪問や日本文化への深い関与を示す情報はなし。「世界のコンフォートフード」を探求するアプローチの一環として日本の要素を取り入れている程度。
